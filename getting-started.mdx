## Welcome to MCWeb!

Thanks for joining our Minecraft server! This guide will help you get started and make the most of your experience.

## Server Connection

**Server Address:** `mcweb.dyplay.at`  
**Version:** Java Edition 1.20+  
**Port:** 25565 (default)

### How to Connect

1. Open Minecraft Java Edition
2. Click "Multiplayer"
3. Click "Add Server"
4. Enter the server address: `mcweb.dyplay.at`
5. Click "Done" and then "Join Server"

## First Steps

### 1. Read the Rules
Before you start playing, make sure to read our [server rules](/rules). Following these rules ensures a fun experience for everyone.

### 2. Find a Spot to Build
- Use `/rtp` to teleport to a random location in the wilderness
- Look for a nice area away from other players
- Consider proximity to resources and biomes you like

### 3. Claim Your Land
Protect your builds using our land claiming system:
- Use `/claim` to claim the area you're standing in
- Learn more about [land claims](/features/claims)

### 4. Start Building
- Gather resources and start your first build
- Check out our [building guides](/guides/building) for inspiration
- Use `/sethome` to set your home location

## Essential Commands

Here are the most important commands for new players:

- `/spawn` - Return to the server spawn
- `/home` - Teleport to your home (set with `/sethome`)
- `/rtp` - Random teleport to find a place to build
- `/claim` - Claim the chunk you're standing in
- `/help` - View all available commands

For a complete list of commands, visit our [Commands Guide](/guides/commands).

## Getting Help

Need assistance? Here are the best ways to get help:

- Ask in chat - our community is friendly and helpful
- Join our Discord server for real-time support
- Contact server staff if you encounter any issues
- Use `/help` in-game for command assistance
